Адрес 0x7000 N
---------------------------------------------------------------------
1.0 Запрос идентификации
	0x7000  	
	------		
	1.Ответное слово (принятая команда)
	2.Номер прибора
	3.Версия

1.1 Запрос специальных параметров
	0x7010  	
	------		
	1. Ответное слово (принятая команда)
	2. Частота генератора(Гц), 
	3. Количество временных окон(шт), 
	4. Длительность временного окна(мкс), 
	5. Уровень дискриминации МЗ(у.е), 
	6. Уровень дискриминации БЗ(у.е),

1.2 Запрос основных параметров
	0x702x  	(бит 0 - включение генератора)
	------		
	1. Ответное слово (принятая команда)
	2. Количество вспышек(ushort)
	3. Наработка генератора(мин)(ushort)
	4. Температура в приборе(0.1 гр)(short)
	5. Количество временных окон(шт)
	6. Длительность временного окна(мкс)
	7..x. Спектр МЗ(ushort)
	x..y. Спектр БЗ(ushort)

---------------------------------------------------------------------
2.0 Установить параметры
	0x7080 	     	
	0xpppp: 0x1 - номер прибора, 
		0x2 - частота телеметрии (1:20833Hz, 2:41666Hz,3:62500Hz,4:83333Hz,)	
	0xdddd: данные
	------
	1. ответное слово

2.1 Установить спец параметры
	0x7090 	     	
	0xpppp: 0x1 - Частота генератора(4..50 Гц), 
		0x2 - Количество временных окон(2..1024 шт), 
		0x3 - Длительность временного окна(2..2048 мкс), 
		0x4 - Уровень дискриминации МЗ(у.е), 
		0x5 - Уровень дискриминации БЗ(у.е),
	0xdddd: данные
	------
	1. ответное слово

2.2 Установить ещё какие-то параметры
	0x70A0 	     	
	0xpppp:	0x1 - Наработка генератора(мин). Используется для сброса в 0
	0xdddd: данные
	------
	1. ответное слово


2.3 Сохранить все параметры в памяти прибора
	0x70F0
